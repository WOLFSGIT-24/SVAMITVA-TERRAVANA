# Svamitva Terravana

A luxury real estate landing page for Svamitva Terravana — Earth Villa community on Kanakapura Road, Bangalore.

Built with **Astro + React + TypeScript + Tailwind CSS + Framer Motion**.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:4321](http://localhost:4321) in your browser.

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- [Astro](https://astro.build) — Static site framework
- [React 18](https://react.dev) — UI components
- [TypeScript](https://typescriptlang.org) — Type safety
- [Tailwind CSS](https://tailwindcss.com) — Styling
- [Framer Motion](https://www.framer.com/motion) — Animations
- [shadcn/ui](https://ui.shadcn.com) — UI component library
- [Zustand](https://zustand-demo.pmnd.rs) — State management
